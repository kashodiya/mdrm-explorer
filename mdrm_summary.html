Micro Data Reference Manual (MDRM) System Summary

Overview

The Micro Data Reference Manual (MDRM) is a catalog of micro and macro data collected from depository institutions and other respondents by the Federal Reserve System. The data are organized into reports, or data series, and consist primarily of financial and structure data.

The MDRM documents the labels and values associated with each data item and is designed to assist end users of the microdata.

Key Concepts

MDRM Identifier

An MDRM Identifier is an eight-character combination consisting of:

  • Mnemonic (first 4 characters): Indicates the Reporting Form or overall name of the data collection
  • Item Code (second 4 characters): Indicates what the specific item is

Example: SVGL2170 refers to total assets on the Thrift Financial Report of Condition.

Cross-Series Consistency

The same data item always has the same Item Code across different series. For example, Total Assets is always assigned the number 2170 across different series (RCON2170, SVGL2170, BHCK2170).

MDRM Manual Structure

The MDRM is broken into two main sections:

1. Reporting Forms and Mnemonics

This section includes links to the reporting forms and a brief description of each reporting series. It provides a link between the reporting forms' public name and the MDRM mnemonic.

URL: https://www.federalreserve.gov/apps/mdrm/series

2. Data Dictionary

This is the most heavily used portion of the MDRM. The Data Dictionary defines each data item and shows all series on which the item appears. It also provides information regarding when the items were available and item descriptions.

Conclusion

The MDRM system provides a standardized way to identify and track financial data items across different reporting forms and institutions. It serves as a comprehensive reference for financial data reporting standards used by the Federal Reserve System.

The system's design allows for consistent tracking of financial metrics across different institutions and reporting forms, with the same item codes being used across different mnemonics to facilitate comparisons and analysis.
